# PWA Icons

This directory should contain the following icon files for the Progressive Web App:

## Required Icons:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png
- icon-192x192-maskable.png
- icon-512x512-maskable.png

## Shortcut Icons:
- shop-96x96.png
- orders-96x96.png
- cart-96x96.png

## How to Generate Icons:

You can use online tools like:
1. https://realfavicongenerator.net/
2. https://www.pwabuilder.com/imageGenerator

Or use a design tool to create icons with:
- Background: Dark green (#0a3d0a)
- Icon: Golden/amber accent (#FBBF24)
- Symbol: Farm/agriculture related (leaf, plant, tractor)

For maskable icons, ensure the safe zone (80% of canvas) contains the main icon content.
