# Changelog

All notable changes to Smart Farming 360 will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2024-03-21

### Added
- ✨ Formspree integration for all forms
- 📧 Contact page with full form functionality
- 📬 Newsletter subscription in footer
- 🎨 Branded form styling with app theme
- 🔄 Form submission redirects with success messages
- 📱 Responsive contact page design
- 🌐 Social media links in contact page

### Changed
- 🎨 Updated footer links to point to contact page
- 📝 Improved form validation and error handling
- 🔄 Enhanced redirect flow after form submissions

### Fixed
- 🐛 Fixed home page scrolling performance issues
- ⚡ Optimized parallax effect (0.5x → 0.3x multiplier)
- 🎯 Added requestAnimationFrame for smooth 60fps scrolling

## [1.0.0] - 2024-03-21

### Added
- 🎉 Initial release of Smart Farming 360
- 🔐 JWT-based authentication system
- 👥 Role-based access control (Admin, Farmer, Consumer)
- 🛒 Complete shopping cart functionality
- 📦 Product management system
- 💰 Ghana Cedis (GH₵) currency integration
- 🌾 35+ products across 7 categories
- 🖼️ 44 product images
- 📊 Admin dashboard with full CRUD operations
- 🌱 Farmer dashboard for product management
- 🏠 Beautiful home page with animations
- 🛍️ Shop page with filtering and search
- 🛒 Shopping cart with real-time updates
- 📱 Fully responsive design
- ⚡ 60fps animations and smooth transitions
- 🎨 Modern UI with glassmorphism effects
- 🌍 Ghana localization throughout
- 📝 Comprehensive documentation
- 🧪 37 passing unit tests
- 🚀 Automated startup scripts (Windows & PowerShell)

### Backend Features
- Express.js REST API
- SQLite database with sql.js
- JWT authentication with refresh tokens
- Password hashing with bcrypt
- Input validation with express-validator
- Error handling middleware
- CORS and Helmet security
- Structured logging

### Frontend Features
- React 18 with TypeScript
- Vite for fast builds
- React Router for navigation
- Context API for state management
- Axios for API calls
- React Toastify for notifications
- Font Awesome icons
- Plus Jakarta Sans typography
- Custom CSS with animations
